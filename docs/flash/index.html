NanoFL has a FlashImporter plugin to convert Flash `fla/xfl` document into NanoFL documents. Most features are well supported (known issues).

Next table show conversion result FLA => NFL=> PNG.

PNG images produced in batch mode by next command:

nanoflc.exe NAME.fla -resize-fit 300x200 -export NAME.png

Sometimes images may have strange position/size - it is normal (this mean strange position/size in FLA).
