ScaleMode

You can set scale mode for your document at File / Properties menu.

noScale

Center viewport, don't scale.

fit

Center viewport, scale, keep aspect ratio, no cropping.

fill

Center viewport, scale, keep aspect ratio, cropping.

stretch

Scale, don't keep aspect ratio.

custom

Don't do anything.
