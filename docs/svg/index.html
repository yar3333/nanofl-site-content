SVG samples

NanoFL has a SvgImporter plugin to convert SVG images into NanoFL documents. Next table show conversion result SVG => NFL=> PNG.

PNG images produced in batch mode by next command:

nanoflc.exe NAME.svg -resize-fit 300x200 -export NAME.png

Sometimes result images may have strange position/size - it is normal (this mean strange position/size in SVG).
